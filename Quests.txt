Приоритеты:

1. Основная задача - представить дерево, которое будем получать с сервера.

Реализовано: 
* Вход+Регистрация.

Необходимо реализовать:

* Представление дерева ( то есть его изображение )

* Сохранение дерева

* Алгоритм роста дерева (+)

Все ещё главными задачами остаётся :

1) Пиксельная картинка типа Bitmap ( представление )
2)Реализовать регистрацию нового пользователя.(Создание нового дерева)

2. Доп. функционал

* Увеличение скорости роста

* Удаление ненужных ветвей

* Сохранять деревья пользователей и востанавливать при ребуте сервера.

* Если будет происходить регистрация пользователя на тот же Логин, необходима ошибка
